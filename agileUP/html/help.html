To use this product fully you must:

  1. Allow blocked content.  The menus are written using Javascript, and that some security products will try to block active content such as this in your browser.
  2. Connect to the Internet.  There are many links to external sites, so you'll want to be connected if you want to read some of the referenced information.

There are various ways to obtain more information regarding Agile UP:

  1. Internal process support.
  2. Training.  Consider sending your project team members on the two-day Agile UP Workshop.
  3. Services.  Scott Ambler has helped a variety of organizations adopt the software process(es) which are best suited for them, and has mentored and trained project teams in the various techniques.

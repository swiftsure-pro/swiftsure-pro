Artifact: Measurement Plan (PSM)
The measurement plan defines which information needs are applicable to a particular project, how the project will be measured to satisfy those information needs, and how the meausrement process will be resourced and managed..
Domains: project_management
Purpose

The Artifact: Measurement Plan (PSM) specifies what base measures should be collected and what derived measures and indicators should be generated during the project to monitor progress, relative to a set of specified project information needs (see Concept: Measurement (PSM)). It is used to collect information on the project as input to the periodic Status Assessment (see  Work Product: Status Assessment).

Representation Options

The Measurement Plan may physically be part of the more general  Work Product: Software Development Plan artifact if the measurement program is a simple one. It may be possible to make use of existing measurement plans that have been used and proven effective.

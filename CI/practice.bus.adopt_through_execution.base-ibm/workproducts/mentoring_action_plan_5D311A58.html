Description
Main Description

This document is an implementation plan for introducing new software development capabilities to a team. It provides estimates for the schedule, project training, and mentoring.

The Mentoring Action Plan describes all important decisions regarding the proposed mentoring approach.
Brief Outline

This artifact typically contains the following sections:

  • Adoption goals,
  • Adoption risks,
  • Deployment plan,
  • Measurement plan,
  • Hardware and software topology to support proposed improvements,
  • Training and mentoring plan,
  • Optional Migration plan,
  • Communication plan
Key Considerations
This document is updated at key stages of the project to address changing circumstances of the project and evolving requirements.

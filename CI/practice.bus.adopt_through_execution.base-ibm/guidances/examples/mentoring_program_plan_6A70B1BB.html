Main Description

In this simple example, a Center of Capability mentoring program is established to accelerate the adoption of the Requirements Management Practice and IBM Rational RequisitePro by several business units in an organization.

It includes the following key plan details:

  • Plans for generating demand for the capability improvements
  • Pilot project assessment plans
  • High level deployment plans
  • Measurements to gather

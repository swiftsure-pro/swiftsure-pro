Main Description

The method library is a collection of process-related assets maintained by the organization for use by project teams. It contains method assets packaged as plug-ins, practices, and configurations. Examples include:

  • Defined processes (for example: lifecycle models and reusable capability patterns)
  • Work products descriptions
  • Role descriptions 
  • Task descriptions 
  • Guidance (for example: templates, checklists, tailoring guidelines)

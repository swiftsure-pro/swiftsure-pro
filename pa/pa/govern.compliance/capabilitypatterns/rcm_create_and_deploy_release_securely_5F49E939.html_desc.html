Capability Pattern: Create and Deploy the Release Securely
This capability pattern covers the activities that will securely produce a software release. It illustrates the "Three Points of Control" strategy in the context of RUP.
DescriptionWork Breakdown StructureTeam AllocationWork Product Usage
Scope
This pattern is for projects that have project delivery process links into a release management process.
Relationships
Context
  • Compliance
Description

This pattern, or set of activities, groups together those activities that are required to securely deliver a release. The scope for this pattern includes:

  • Developing software
  • Packaging software
  • Making sure that baselines satisfy the requirements of the Distribution Policy
  • Distributing the software release package

Alternatives

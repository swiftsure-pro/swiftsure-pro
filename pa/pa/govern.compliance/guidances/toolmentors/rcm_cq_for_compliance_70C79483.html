Tool Mentor: ClearQuest for Compliance
This tool mentor describes the ways in which IBM Rational ClearQuest can support compliance enhancement projects in the area of configuration and change management.
Main Description

ClearQuest contributes to the configuration and change management aspects of the IBM Rational Software Delivery Platform process solution. It provides the ability to deliver an auditable change by supporting management and control of software assets through workflow automation. Automated workflow management can help enforce process, audit trails, electronic signatures, and reporting, all of which allow organizations to document authorizations and signoffs at key stages in the lifecycle.

The change management aspects of the IBM Rational Software Delivery Platform process solution support the ability to initiate a ecompliance enhancement project with the appropriate levels of traceability and control.

The table below highlights some of the key aspects required, and shows how ClearQuest aligns with these needs.

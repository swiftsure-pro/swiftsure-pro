Description
Main Description

This artifact states the procedural rules and quality standards required to authorize the distribution of a release. It also enumerates the main releases planned for a project and specifies what additional rules apply to each of them.

An example of a procedural rule for distributing packages might be:

Distribute only to test servers (for a particular release)

    Main Description

    Compliance concerns

    The key aspects of compliance management that have an impact on the responsibilities of the Integrator include:

    • Baselines
      • Adopting a standard baseline establishment strategy that is documented and followed
      • Implementing promotion levels (the aim is to be able to clearly define the status of a baseline and provide security regarding what can be done with it)
      • Using electronic signatures to provide a formal approval of baselines
    • Validating change requests
    • Creating, maintaining, and executing the build script, as well as exercising smoke tests

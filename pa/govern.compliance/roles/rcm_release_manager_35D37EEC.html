Role: Release Manager
This role is responsible for collating the released work products from multiple projects to produce a release that can be delivered into the production environment.
Synonyms: Release Engineer

    Main Description

    The Release Manager owns the release management process and has the responsibility and authority for the overall process results. The Release Manager is responsible for the overall quality of the process, is the main coordinator within this process, and the focal point regarding releases for both the customer and the IT organization. Therefore, all managers in the IT organization must support the person in this role.

    The Release Manager has the following responsibilities:

    • Establish measurements and targets to improve development process effectiveness and efficiency
    • Identify and manage critical success factors
    • Define Release Management team roles, responsibilities, and accountabilities
    • Enforce the release management process across the project
    • Report release status and progress to peers and executives
    • Facilitate, resolve, or escalate cross-functional issues
    • Audit release management process execution
    • Be accountable for the cost and investment in the release management process
    • Represent the release management process to all external groups
    • Control release distribution
    • Perform release management assessment and ensure implementation of the release management strategy
    Staffing
    Skills

    The following skills are recommended for the person who fills the Release Manager role:

    • Experience in the software development lifecycle, the domain of the application, and platform
    • Experience with release management processes
    • Experience in the production environment for which the role is responsible
    • Ability to manage the delivery of multiple projects
    • Leadership and team-building capabilities
    • Customer management capabilities 
    • Experience as the interface with the IT Operations team for internally facing applications
    • Experience as the interface with the Manufacturing team for externally facing products
    Assignment ApproachesThis is typically a role that heads up an independent Release Management team that controls delivery of all software applications into the production environment. Other roles within this team may include the Release Tester and the Release Distributor.

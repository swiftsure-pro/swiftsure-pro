Role: Technical Lead
This is a senior Designer/Implementer for an application. This role has both technical and some managerial responsibilities. This role guides and oversees development of application software components.
Synonyms: Application Lead, Lead Developer
Role Sets: developers

    Main Description

    Technical Leads manage the work of Developers within an application development project. Their key responsibilities are:

    • Assign tasks and distribution of work amongst Designers, Implementers, and Integrators.
    • Approve required artifacts and actions
    • Contribute to the design of the schema with IT controls

    Staffing
    Skills

    The following skills are recommended to fulfill the Technical Lead role:

    • Experience in the software development lifecycle, the domain of the application and platform
    • Experience with the application for which the role is responsible, including understanding the architecture, key mechanisms, and the design of the application
    • Knowledge of the development team and their skill sets
    • Leadership and team-building abilities
    Assignment ApproachesFor smaller projects, a single person can act as Project Manager and also take on this technical leadership role.

Main Description

Assembled in this category are:

  1. The roadmap on how to use RUP enhanced for compliance management (RCM) to develop Application Controls in an auditable manner
  2. A collection of subcategories describing key additions to RUP disciplines to support auditable Application Control Development. These include:
  • Roles
  • Work products
  • Tasks
  • Changes to discipline workflows
  • Fundamental guidance

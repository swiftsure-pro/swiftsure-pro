Description
Main Description

This artifact provides the detail of the change workflow that the project will follow. It defines the record types, the states that they go through, and the associated attributes. It clearly identifies the control points and their implementation into the workflow.

The information contained in this work product is a part of the  Configuration Management Plan. It has been separated into a sub-artifact because its content is of critical importance in documenting compliance to an auditor.

Key Considerations
This specification defines the schema for the associated change control tool. Many organizations have a central tooling team that handles this task on behalf of the project.
Tailoring
Reasons for not needingThis artifact may be enclosed as part of the  Artifact: Configuration Management Plan. However, this information is essential for any kind of audit or inspection, so it is useful to draw this artifact out as a key piece of the Configuration Management plan.

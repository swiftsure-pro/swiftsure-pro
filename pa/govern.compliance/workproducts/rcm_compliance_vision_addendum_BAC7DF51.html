Description
Main Description

This compliance-focused work product highlights those features of the process and of the developed system that contribute to achieving compliance objectives. The addendum does not contain details of the techniques, metrics, and so on, but provides a summary of higher-level information:

  • The compliance requirements and goals to be addressed by the project
  • The approach to achieving them and reaching Compliant status

This artifact should be maintained throughout the development lifecycle.

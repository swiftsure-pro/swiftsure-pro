Outcome: Release Control Point
This work product is a record of the approval of a product for distribution, either into a quality assurance environment or production.
Domains: deployment
Work Product Kinds: Record
Extends: Control Point
Purpose

To ensure control over the delivery of a set of build objects into the QA and production environments.

Main Description

The Release Control Point provides a gate that prohibits deployment of a baseline to QA or production if it is not registered as QA-Ready or Production-Ready, respectively.

This approval will specify the list of versions of component products and subsystems approved for distribution (usually represented as a Bill of Materials).

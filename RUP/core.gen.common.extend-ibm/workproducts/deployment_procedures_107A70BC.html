Description
Main Description

This include procedures for:

  • installing user support materials (e.g documentation)
  • setting up user definitions, access rights, passwords
  • initializing data sources
  • including the system release and new users in support and recovery procedures
  • confirming that new locations and users are fully operational
Tailoring

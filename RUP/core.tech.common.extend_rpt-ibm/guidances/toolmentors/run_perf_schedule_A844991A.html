Tool Mentor: Run a Performance Schedule Using IBM Rational Performance Tester
This tool mentor describes how to use IBM Rational Performance Tester to run a performance test schedule.
Tool: IBM Rational Performance Tester
Relationships
Related Elements
  • Develop Test
Main Description

Overview

When you run a schedule, IBM® Rational® Performance Tester automatically sets up a typical launch configuration. A test runs on the local computer with one user. A schedule runs with the user groups and the locations that you have chosen. However, the execution results have a default name (the same as the schedule or test, with a suffix), and are stored in a default location. You can also run a configured schedule.

Tool Steps

To run a schedule, perform the following steps:

  1. In the Test Navigator, select the given schedule
  2. Right-click the schedule, and then click Run > Run Performance Schedule

To run a configured schedule, perform the following steps:

  1. In the Test Navigator, expand the project until you locate the schedule
  2. Right-click the schedule, and then click Run > Run
  3. In the Configurations pane on the left, click Performance Test Schedule and then click the name of the configured schedule you wish to run
  4. Click Run

For more information on running a performance schedule in Rational Performance Tester, see the topics Running a configured schedule andSetting a Launch Configuration in the online Help.

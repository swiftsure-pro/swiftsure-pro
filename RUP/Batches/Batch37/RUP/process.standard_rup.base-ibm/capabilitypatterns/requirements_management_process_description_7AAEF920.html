Main Description

The Requirements Management Process Description describes the requirements artifacts, requirement types, their respective requirements attributes, how requirements traceability will be managed, and specifies the information to be collected and tracked. It also specifies how requirements changes will be managed, as well as addressing requirements organization, tool automation, the necessary reports, views, and queries that will be generated and the related processes for requirements and requirements change management.

Task: Resolve Change Request
This task indicates that a change has been implemented as requested and is ready for verification.
Disciplines: Configuration and Change Management, Project Management
Purpose
The purpose of this task is to resolve a change request.
Relationships
RolesPrimary Performer:

    		Optional:

    Outputs

      Main Description

      This task describes the steps to indicate that a change has been implemented and is ready for verification.

      Hints and tips

      • Review the resolution of the change request to ensure it meets all stakeholder expectations.
      Steps
      Verify artifacts associated with the change request
      Verify all required artifacts against the change request, and that they are in the appropriate state.
      Key Considerations
      Verify that all required Change Sets are identified against the Change Request, and that they are in the appropriate state.

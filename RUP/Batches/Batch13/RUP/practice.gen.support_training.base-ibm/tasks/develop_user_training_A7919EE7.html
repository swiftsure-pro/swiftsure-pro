Task: Develop User Training
To create the end user training materials
Disciplines: Deployment
Purpose
To create or extend the end user training materials that are necessary for running a class and therefore for meeting the objectives set for the training. End user training materials are also necessary to help the instructors prepare for the class.

      Main Description
      The End User Training Materials are the materials used by students and instructors on a course or training session.

      They contain the following, for each of the training modules:
      • Foils, hand-outs, instructor guides, or just relevant instructor notes.
      • Exercises.
      • Exercise data in the training system.
      • Agenda for the course-module, indicating the topic sequence, where which exercises fit in and the duration of the different units.

      End User Training Materials also include the instruments and the tools to evaluate and measure the training material, specifically:

      • Who will collect the data and how
      • How the measurement data will be processed.
      • Measurement instruments and tools (selected or developed).
      • Target values for the evaluation levels.

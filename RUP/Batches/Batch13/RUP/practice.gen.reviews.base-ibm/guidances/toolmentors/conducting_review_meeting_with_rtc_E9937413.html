Main Description

Reviewing work items

Once a reviewer has been requested to review a work item, they may review the work item, leave comments, and either reject or approve the work item. 

[Add info about shared query]

1. Locate the work item to review.
2. Conduct your review by looking through the Description and the discussion.  If a file has been added to review you can find it in the links tab.
3. Once you have decided to approve or reject the work item, select the approvals tab and find your name and change the State from Pending to Approved or Rejected.

For a work item to pass the review all of the reviewers must approve it.  If only one reviewer rejects the work item, it is rejected.

Tool Mentor: Execute Test using CQTM
This tool mentor describes how to execute tests using IBM-Rational CTQM.
Tool: IBM Rational ClearQuest Test Management

Main Description
CQTM provides you with the flexibility to execute your tests in different ways depending on your specific test environment needs.
You can execute a test script in three different ways:
  • Execute a configured test case or test suite record from the IBM® Rational® ClearQuest® Client or the Rational ClearQuest Client for Eclipse.

    The Rational Test tool or supported Eclipse Test and Performance Tools Platform (TPTP) tool must be installed on the system where the configured test case or test suite record is executed.

  • Execute from a Rational Test tool or supported Eclipse Test and Performance Tools Platform (TPTP) tool and import the test log files by using the Rational ClearQuest Client or the Rational ClearQuest Client for Eclipse.

    The supported test tool can be executed on any machine but the location of the test log file must be defined as a Rational ClearQuest Test Manager file location.

  • Execute from an unsupported test tool from any machine.

    Using any Rational ClearQuest client, you can manually create a test log record and then associate with a configured test case record.

For more information about executing tests, see the Help system for the Rational ClearQuest Client or the Rational ClearQuest Client for Eclipse.


Using Rational Test tools with Rational ClearQuest Test Manager

You can execute test scripts with any of these Rational Test tools:

  • Rational Functional Tester (RFT) version 6.1.1.1
  • Rational Performance Tester (RPT) version 6.1.2
  • Rational Manual Tester (RMT) version 6.1.1.1

Rational ClearQuest Test Manager also supports the Eclipse Test and Performance Tools Platform (TPTP) test tools: Junit, URL Test, and Manual Test.

To use any of the supported testing tools with Rational ClearQuest Test Manager, the tool must be installed on the same machine as a Rational ClearQuest Client or a Rational ClearQuest Client for Eclipse.

Rational ClearQuest Test Manager supports two options for executing test scripts for supported test tools:
  • A configured test case or test suite record can be executed from Rational ClearQuest Test Manager and the results are displayed on the Execution page of the Test Manager view or the Test Results view. The test results can be committed and written to the database.
  • A test script can be executed directly from a Rational Test tool and the resulting test log file can imported into the Test Results view. The test results can be committed and written to the database.

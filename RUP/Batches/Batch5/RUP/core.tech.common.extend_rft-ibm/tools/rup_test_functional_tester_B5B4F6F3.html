Tool: IBM Rational Functional Tester
The IBM Rational Functional Tester tool is an object-oriented automated testing tool capable of testing Java, HTML, VB.NET and Windows applications.

Main Description
IBM® Rational® Functional Tester is an automated testing tool that allows you to record scripts using an object-oriented recording technology. It is offered in several development environments: Rational Functional Tester, Java™ Scripting and Rational Functional Tester, VB.NET Scripting.

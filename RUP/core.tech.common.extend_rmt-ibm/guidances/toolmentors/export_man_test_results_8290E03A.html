Main Description

Overview

One of the essential tasks of any test effort is the identification and capture of test results. To capture test results, you need some systematic way to log results from a test run. Once you have completed running tests, you may export the results in the form of an execution log export. IBM® Rational® Manual Tester allows you to easily export results. You can use the Export Wizard to export your execution log files.

Tool Steps

To export test results, perform the following steps:

  1. Select File > Export
  2. Use the Rational Manual Tester Execution Results Export Information Wizard to export the execution log to a .csv file
  3. In the wizard, add the execution log(s) you want to export
  4. Ensure that the path for the export is the desired location
  5. Select the CSV Formatter
  6. In the wizard, click Finish

The process of reviewing and evaluating test results is made easier by the availability of a Test Results Summary export. With Rational Manual Tester, you are able to include only the log(s) you are interested in analyzing.

For more information on exporting test results, see the topics Exporting Log Files and Viewing Test Results in the Log in the Rational Manual Tester online Help.

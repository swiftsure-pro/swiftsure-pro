Role Descriptor: System Administrator
This role maintains the development infrastructure, both hardware and software. This includes installation, configuration, backup, etc.
Based on Method Role: System Administrator
Relationships
Performs:
  • Implement System Management Procedures
  • Initialize System Data Sources
  • Install System Components
Responsible for:
  • Data Migrated
  • Production Environment Established
  • System Management Environment Established
Additionally Performs:Modifies:
  • Data Migrated
  • Production Environment Established
  • System Management Environment Established
Main Description

An individual taking on the role of System Administrator needs a good understanding of the specific hardware and software components used on a project, and the possible dependencies between these components.

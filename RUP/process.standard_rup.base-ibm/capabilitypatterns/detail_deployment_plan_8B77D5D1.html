Main Description

Use the chosen form of review process (such as a walkthrough or formal inspection) to review the Deployment Plan to ensure that it:

  • Identifies and establishes the chronological sequence in which the new system will be installed at each location.
  • Establishes a training schedule for site personnel.
  • Creates the site survey plan and checklist.
  • Creates the installation verification procedures that will be executed prior to the cutover of each site.
  • Identifies the deployment responsibilities of each participating person, organization, subcontractors, and vendor.
The Deployment Plan should only be reviewed at each increment and will probably require little or no change.

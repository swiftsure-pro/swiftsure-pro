Description

This pattern, or set of activities, is typically repeated for every build in an iteration.  The frequency of build production varies from project to project.  Some projects build on a daily basis; others produce one build per iteration.

[within Scope] in the name of this pattern indicates that the activities within this pattern are bounded by a particular scope, such as a particular component, for example.  This pattern is intended to be used multiple times, once for each appropriate 'scope'.  For example, once for each component in an iteration.

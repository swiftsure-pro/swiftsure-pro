Tool Mentor: Review Test Results using CQTM
This tool mentor describes how to review and commit results using IBM-Rational CQTM.
Tool: IBM Rational ClearQuest Test Management

Main Description

After executing a configured test case or test suite record, you can review and commit test results by using the IBM® Rational® ClearQuest® Client or the Rational ClearQuest Client for Eclipse.

When you commit the test results, a test log or suite log record is created and associated with the executed configured test case or test suite record.

You can use any client to access a test log or suite log record from the associated configured test case or test suite record. You can also use a query.

For more information about test results, see the Help system for the Rational ClearQuest Client or the Rational ClearQuest Client for Eclipse.

Main Description

Using motivators with test plan and test case records

A motivator provides the rationale for some or all of the testing. In Rational ClearQuest Test Manager, this information can be provided as a test motivator file or a Rational ClearQuest requirement.

A test motivator file is an external file that provides details about a test plan or test case. For example, a test motivator file could be a Word document or Excel spreadsheet containing detailed information about a specific test plan or test case. You can reference a test motivator file from a test plan and test case record using a Rational ClearQuest Client or a Rational ClearQuest Client for Eclipse. Before you can associate a test motivator file with a test plan or test case record, you must create a file location record to define the path to the test motivator file.

If you are using the Rational RequisitePro integration version 1.8 with Rational ClearQuest or the Rational ClearQuest Enterprise schema version 7.0, you can use any Rational ClearQuest client to associate a Rational ClearQuest requirement record with a test plan, test case, or configured test case record.

You can use associations between test plan and test case record with a test motivator to determine:
  • If there is test coverage for the motivator
  • If a test plan associated with a motivator has an associated external file (Excel or Word document)

For more information about test motivator files, see the Help system for the Rational ClearQuest Client or the Rational ClearQuest Client for Eclipse.

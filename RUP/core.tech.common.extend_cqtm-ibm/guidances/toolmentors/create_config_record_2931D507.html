Main Description

Overview

Before creating a configuration record, determine which configuration platform details have a testing impact. You must create a configuration record for each testing platform on which a test is run.

Before creating a configuration record, you must create configuration attribute and configuration value records.  For more information, see the CQTM online Help topics Creating a configuration attribute record and Creating a configuration value record.

Tool Steps  

To create a configuration record:
  1. Log in to a database that supports IBM® Rational® ClearQuest® Test Manager.
  2. Using a supported Rational ClearQuest client:
    1. From a Rational ClearQuest for Windows client, click Actions > New and from the Choose Record Type window, click TMConfiguration and then click OK.
    2. From a Rational ClearQuest Web client, click New and then select TMConfiguration and click Go.
  3. From the Configuration page:
    1. Type a configuration name in the Name field.
    2. Type an optional description in the Description field.
    3. Add a value to the Configuration Attribute List by clicking Add and then selecting one or more configuration values from the Browse Record Type window.
  4. In the Rational ClearQuest for Windows client, click OK. In the Rational ClearQuest Web client, click Save.
After creating a configuration record, you can associate it with a configured test case or test suite record.

For more information, see the CQTM online Help topic Creating a configuration record, Using a configuration record, and Modifying a configuration record.

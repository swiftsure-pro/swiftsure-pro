Main Description

Overview

Before you start using IBM® Rational® ClearQuest® Test Manager, you must first create an asset registry record. An asset registry represents a testing effort. This can take the form of a product, a release, or a product release.

Tool Steps

To create an asset registry record:
  1. Log in to a database configured for Rational ClearQuest Test Manager.
  2. Using a supported Rational ClearQuest Client:
    1. From a Rational ClearQuest for Windows client, click Actions > New and from the Choose Record Type window, click TMAssetRegistry and then click OK.
    2. From a Rational ClearQuest Web client, click New and select TMAssetRegistry and click Go.
  3. Type a name in the Name field.
  4. Type a description in the Description field.
  5. In the Rational ClearQuest for Windows client, click OK. In the Rational ClearQuest Web client, click Save.
You must create an asset registry record before you can create an iteration record. When you create a iteration record, you must associate it with an existing asset registry record.

For more information, see the CQTM online Help topics Creating an asset registry record, Using an asset registry record, and Modifying an asset registry record.

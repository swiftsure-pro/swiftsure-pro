Setting up work items for review 

Rational Team Concert allows you to set work items for approval, review, or verification.  In this tool mentor we will focus on reviews, however the approval and verification feature are very similar. A work item may be a feature, task, enhancement, track build item, risk, use case, or scenario.  Setting up a review for any of these work item types is the same.

  1. Open the work item you want to have reviewed.
  2. Select the Approvals tab.
  3. Click the New Approval button.
  4. Select the type of Approval, for this example select Review.
  5. Change the Subject to correspond with the type of approval, for this example we will use Review <task name>yymmdd.  This way you can create multiple reviews for the same task and keep track of each review.
  6. Optionally include a due date.
  7. Select Add Approver...
  8. If the reviewers are not listed in the Matching Users field, enter part of the name of the person you would like to add and click Search.  You may select multiple users by using ctrl and clicking their name.
  9. Once you have all the reviewers selected click Ok.
Now you have successfully set up your work item to be reviewed.  Notify the reviewer(s) of the work items to review.  If the reviewer(s) can view the work item by typing in the the work item number on the bottom left, or do a search for all of the work items which need reviewing by selecting "Pending approvals for me" under the shared queries folder.

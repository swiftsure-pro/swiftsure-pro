Main Description

IBM® Rational® Functional Tester Extension provides the ability to automate the testing of MVS, VM, OS/400 and Unix applications by using the Host Access Class Library (HACL) terminal emulator to connect as a TN3270, TN5250 or VT terminal. FTE can be installed and run on Windows 2000, Windows XP and Windows 2003 Server platforms. The test scripts FTE helps RFT create can be either Java scripts (in the Eclipse development environment) or Visual Basic scripts (in the Visual Studio .NET environment).

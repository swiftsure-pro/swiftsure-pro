Role Descriptor: Database Specialist
This role develops and maintains database systems used for storing, analyzing, and retrieving data.

Main Description
This role is responsible for outlining and detailing the logical data model, as well as developing the physical database design. This role also describes how to migrate legacy data source to a target database.

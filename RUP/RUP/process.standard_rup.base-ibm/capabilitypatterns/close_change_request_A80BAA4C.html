Task Descriptor: Close Change Request
This task closes a verified change request.
Based on Method Task: Close Change Request

Steps
Review content of change request
Review the content of the change request to ensure that the implementation and verification are complete. Ensure that any supporting information required to objectively prove that the request can be closed is referenced. Gain concurrence from the Submitter of the change request that the request can be closed.
Close the change request

If the results of the review are satisfactory, close the change request. If there is missing information or outstanding issues with the change request, take the appropriate action before closing the change request.

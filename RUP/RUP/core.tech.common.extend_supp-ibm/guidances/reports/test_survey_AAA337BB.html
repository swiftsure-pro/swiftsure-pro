Report: Test Survey
The test survey provides information on the structure and contents of the test implementation.
Relationships
Related Elements
  • Test Suite
Main Description
  1. Introduction - Briefly summarizes the scope of the tests.
  2. Overview of Test Suites - Lists the hierarchy of Test Suites including a brief description for each test suite, and shows how Test Suites depends on or utilize the services of other Test Suites.
  3. Test Suite Outlines - For each Test Suite, enumerates a list of Test Scripts each Test Suite uses and-if the test script names are not sufficient-a brief description of each Test Script.
  4. Test Suite Traceability - Optionally, the report may also show traceability references to which Test Motivator(s) and/ or Test Case(s) each Test Suite addresses.

Tool Mentor: Creating Build Values using CQTM
This tool mentor describes how to create build values using IBM-Rational CQTM.
Tool: IBM Rational ClearQuest Test Management
Relationships
Related Elements
  • Run Tests
Main Description

Overview

In IBM® Rational® ClearQuest® Test Manager, a build value is mandatory. When a configured test case or a test suite record is executed, you are prompted for an associated build value.

Before creating build values, identify the builds that have a testing impact. You must create a build value for each milestone.

Tool Steps

To create build values in the Rational ClearQuest for Windows client:
  1. Click Edit > Named Lists > Build to open the Edit Build window.
  2. Click in the Edit Build window and type a build value.
    1. To add multiple build values, right-click a build value and click Add.
    2. To insert a build value in an existing list of values, select a value in the list and click Insert.
  3. Click OK.
When executing a configured test case or test suite from the Rational ClearQuest Client or the Rational ClearQuest Client for Eclipse, you are prompted for a build value. Any build values entered previously are displayed.

For more information, see the following topics in the CQTM online Help:  Creating build values, Using build values, and Deleting a build value.

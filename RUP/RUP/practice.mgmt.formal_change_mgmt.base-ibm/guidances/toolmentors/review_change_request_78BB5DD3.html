Tool Mentor: Reviewing a Change Request Using Rational ClearQuest
This tool mentor describes how to review a change request using Rational ClearQuest.
Tool: Rational ClearQuest
Relationships
Related Elements
  • Review Change Request
Main Description

Tool Steps

  1. Run a query to identify change requests that need to be reviewed (for example: "Submitted for my Review").
  2. Select the change request by marking its line on the result set grid.
  3. Perform action Analyze. Click Action and then select Analyze. The record state changes to Analyzed.
  4. Enter the required data in the Analysis tab, and optionally additional information on the Notes tab.

Tool Mentor: Resolving a Change Request Using Rational ClearQuest
This tool mentor describes how to resolve a change request using Rational ClearQuest.
Tool: Rational ClearQuest
Relationships
Related Elements
  • Resolve Change Request
Main Description

Tool steps

Without UCM integration

  1. Run a query to identify the change request that needs to be Resolved (for example: "My open CRs").
  2. Select the change request by marking its line on the result set grid.
  3. Perform action Resolve. Click Action, and then select Resolve. The record state changes to Resolved.
  4. In the Resolution tab, enter data in the mandatory fields. Optionally, enter additional information on the Notes tab.

With UCM integration

  1. When you finish your work, deliver your change(s) from IBM® Rational® ClearCase®.
  2. The ClearCase-ClearQuest integration automatically performs the action Resolve. The record displays if there are unfilled mandatory fields.

Hint: You can enter additional information before or after the delivery by selecting the record and performing the action Modify.

Tool Mentor: Closing a Change Request in Rational ClearQuest
This tool mentor describes how to close a change request using Rational ClearQuest.
Tool: Rational ClearQuest
Relationships
Related Elements
  • Close Change Request
Main Description

Tool Steps

  1. Run a query to identify the change request that may be Closed (for example: "Ready to Close").
  2. Select the change request by marking its line on the result set grid.
  3. Perform action Close.Click Close, and then select Close. The record state changes to Closed.
  4. Enter the closure reason (such as Fixed, Not a Bug, Rejected, and so on). Optionally, enter additional information on the Notes tab.
  5. Click OK.

Task: Specify Training Requirements
To create the specification for the user training required for the application under construction
Disciplines: Deployment
Purpose
To create the specification for the user training required for the application implementation. The specification defines the boundaries and relationships between course units as well as
  • Content
  • Audience
  • Purpose
  • Prerequisites
  • Duration
  • Comments, requirements, exercise
  • Description of where and when to perform a specific task.
Relationships
RolesPrimary Performer:
  • Course Developer
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Education and Training Plan
  • Learning Solution Design
    Outputs
    • Education and Training Evaluation
      Main Description
      For each type of media to be used for training the user in the application the boundaries must be specified. The specification identifies the responsibilities of each part from the client and the project team.
      Steps
      Define objectives of the training
      Describe content of each training unit
      For each training unit define rough requirements for training
      For each topic sketch content / scenario of exercises to be included
      Identify likely / most significant errors for each topic and make note of how these can be corrected
      Note User support material that will support each topic
      Determine length of course modules
      Specify equipment and facilities required to support the delivery of the User training

Task: Detail System Management Plan
Update the system management plan and software distribution to make sure they address all of the requirements.
Disciplines: Deployment
Purpose
The purpose of this task is to complete the systems management and software distribution plans.

This provides the basis for the following:
  • The selection of system management tools.
  • The selection of systems management processes.
  • Selecting and defining job descriptions, based on identified roles, to support the IT system.

      Main Description

      Starting from the outline IT Services Strategy developed earlier, develop the plan into a detailed framework adequate to ensure the system management related nonfunctional requirements can be met. Make a list of necessary hardware platforms and software environments that are necessary to implement functionality of the systems. Utilize the Systems Management Framework Design (SMFD) matrices for collection of all such information.

      Also, refine the Software Distribution Plan developed earlier, defining precisely what will be distributed where.

      • Correlate systems management functions to the Deployment Units, Non-functional Requirements, Operational Model, network architecture, and Software Distribution Plan.
      • Systems management evaluation will typically require a database to catalog all the information, so that it can be sorted and reports generated. Manual processing of all the data for a large enterprise with many elements can be unmanageable.

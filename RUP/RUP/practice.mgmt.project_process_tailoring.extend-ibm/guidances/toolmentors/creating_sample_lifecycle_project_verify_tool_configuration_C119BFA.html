Main Description

The Money that Matters sample is a feature rich CLM sample with linked artifacts across RTC, RQM and RRC. Every application contributes to this sample with specific assets from their domain. The goal is to provide as much sample data as a real production environment would have for a real world application and customer.

For instructions on how to set up the environment for this sample application, please refer to the sample instructions on jazz.net (requires account).

Task: Plan Team Preparation
This task focuses on preparing the test team based on specific test project needs.
Disciplines: Test
Purpose

The purpose of this task is:

  • To define the test team preparation objectives
  • To plan for the test team preparation content
Relationships
RolesPrimary Performer:
  • Test Manager
Additional Performers:
InputsMandatory:
  • Test Motivator
    		Optional:
    • [Project Definition and Scope]
    • [Project Risk]
    • [Project Work]
    • Test Plan
      Outputs
      • Test Plan
        Steps
        Determine Test Team Skill Requirements
        Review the test strategies that will be employed to identify test team skills for a particular test effort. Ensure that the right mix of testing roles is achieved based on the test objectives in the Test Strategy.
        Capture Test Team Preparation Requirements
        By discussing the test effort with technical leads and product domain experts, determine specific test team preparation needs. Consider both managerial and technical skills for the given testing effort, and document the test roles and responsibilities in the test plan based on specific test team needs.
        Compile Comprehensive List of Test Team Preparation Objectives
        Based on the test roles and responsibilities captured in the test plan, develop a list of test team objectives for the given test effort. Include this information in the test plan.
        Discuss Test Team Role and Responsibility Assignment with Test Staff
        Clarify test role and responsibility assignment with test staff. Ensure that the given staff member understands the role(s) as well as the function of the role(s) within the test team.
        Review Test Team Preparation Plan
        Meet with technical leads and product domain experts to review the test team planning, and update the test plan to reflect the test team requirements and project staff assignment of test roles and responsibilities. If the test team is part of a larger roles and responsibilities matrix for the project, provide this information to the project manager.

Task Descriptor: Communicate Test Project Status
This task focuses on providing the development team with timely information on the status of the test effort.
Based on Method Task: Comunicate Test Project Status

  • Test Evaluation Summary
Steps
Capture Status Against the Test Plan
Based on periodic reviews of the test plans (both high-level and iteration plans), any variance to the plans are noted. Capture the variance to plan in reports that you can use to describe status to the development team.
Communicate Testing Issues to Development Team
After you have identified test planning issues, discuss these issues with the development team on a regular basis. Process the feedback of these discussions in updates to test reports. Compile a list of major test issues, and discuss these issues with the test team.
Hold Test Effort Review Meetings with Development Team
As the test effort progresses, hold review meetings with the development team to keep the team updated and involved in the test effort. If these meetings address team-generated recommendations for improving the testing, capture these recommendations during the meeting in a report.
Provide Development Team with Test Reports on a Timely Basis
Provide the development team with test reports, test evaluation summaries, defect status reports, test coverage reports, and test environment reports.

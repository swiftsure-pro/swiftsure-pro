Task: Install System Components
Install the integrated subsystems for production.
Disciplines: Deployment
Purpose
Set up the hardware and software infrastructure for production to achieve the objectives of the new application system.
Relationships
RolesPrimary Performer:
  • System Administrator
Additional Performers:
InputsMandatory:
  • None
Optional:
  • Deployment Plan
  • Deployment Procedures
  • Software Distribution Plan
    Outputs
    • Production Environment Established
      Steps
      Install the hardware infrastructure according to the deployment work plan
      Install the software infrastructure and configure according to the plan
      Setup the production environment in which the integrated subsystems need to run
      Install the integrated subsystems, according to the plan
      Key Considerations

      Deployment and cut over of the new system are highly dependent on the contents of the system and the size of the end-user departments.

      This work can involve many departments and subcontractors/vendors.

Task: Outline Deployment Plan
Outline the overall approach to deployment for the scope of the project.
Disciplines: Deployment
Purpose
The purpose of this task is to create a high level plan to guide and control the rollout of the system to all locations.
Relationships
RolesPrimary Performer:
  • Deployment Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • [Project Definition and Scope]
    Outputs
    • Deployment Plan
      Steps
      Propose a potential sequence in which the new system will be installed at each location
      Define the need for installation verification procedures
      Identify the deployment responsibilities of each participating organization, subcontractor, and vendor

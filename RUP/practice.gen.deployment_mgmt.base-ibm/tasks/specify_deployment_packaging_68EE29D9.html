Task: Specify Deployment Packaging
Specify the materials to be packaged as part of the deployed system or product.
Disciplines: Deployment
Purpose
The purpose of this task is to specify all the materials required to deploy and deliver the system or product.
Relationships
RolesPrimary Performer:
  • Deployment Manager
Additional Performers:
InputsMandatory:
  • None
Optional:
  • [Project Definition and Scope]
  • [Technical Architecture]
  • [Technical Specification]
  • Software Distribution Plan
    Outputs
    • Bill of Materials
    • Deployment Packaging
      Main Description

      In addition to the components comprised by the executable work products, consider other items as installation instructions, user help materials, warranty registration cards, and their actual containers (i.e., boxes, envelopes, etc.).

      Ensure all contractually required items for product acceptance are accounted for. Review any customer acceptance plans for descriptions of how the customer will evaluate the deliverables.  Make sure that the materials account for relevant acceptance criteria.

Task: Develop Data Migration Specifications
Describe how to migrate a legacy data source to a target database.
Disciplines: Development
Purpose

To specify the data migration by defining its scope, its data profile, the mapping between the data sources and the target database, and any required data transformation processes.

Both one-time data conversions and ongoing data interfaces are addressed by this task.

        Key Considerations
        It is preferable to have legacy developers from the source data references as participants in the data mapping process. Business analysts who possess an intimate knowledge of the data that needs to be migrated and those familiar with the target systems should also be active participants in the process.

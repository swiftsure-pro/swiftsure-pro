Main Description

The essential construct in the IBM® Rational Quality® Manager (RQM) tool is the test plan. A test plan defined in RQM will drive activity for distributed teams through construction, deployment, execution and analysis phases of the project.

RQM provides the ability to capture your test plan completely and manage your test project within the context of the plan. Universal reports and dashboards can be generated throughout the project and from anywhere, allowing continual governance and feedback for quality objectives. RQM's reporting capabilities ensure that the status of a project and of individual test activities are readily available, and at any time the team can understand what needs to be done next.

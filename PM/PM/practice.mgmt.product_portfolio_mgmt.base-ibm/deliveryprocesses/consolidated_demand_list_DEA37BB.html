Main Description

Information to be tracked relates to approved work that is in execution, and planned and approved work that has not yet started. Depending on average project length, planning horizon and resources availability, this artifact could also track future possible demand, such as planned work not yet approved. This may be necessary in order to identify critical resources that are scarcely available on the market which would require lengthy hiring processes.

Depending on organizational structure and operational focus, this artifact can contain different types of information, but typically it would contain information such as:

  • work pipeline
  • project or project-proposal name
  • required skill name and amount
  • time period when the skill or resource is needed

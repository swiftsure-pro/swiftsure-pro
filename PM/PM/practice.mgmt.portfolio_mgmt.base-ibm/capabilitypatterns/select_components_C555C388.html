Task Descriptor: Select Components
Triage the requests and components being considered for inclusion within the portfolio.
Based on Method Task: Select Components
Main Description

Component selection is the most effective mechanism within the portfolio cycle to narrow the funnel and eliminate the ideas that cannot be self-sustained or demonstrate enough value. 

Conclude the initial triage of all components being considered for inclusion within the portfolio. Compare all components within a given work pipeline against the predefined portfolio metrics, measurements, and thresholds to identify which components meet all of the required parameters and add value to the portfolio.

These are examples of portfolio metrics, measurements, and targets:

  • Business alignment
    • Strategic fit
    • Multiple-criteria scoring models
    • Capacity constraints
  • Financial
    • return on investment (ROI), internal rate of return (IRR), payback
    • net present value (NPV)
    • Cost versus benefit analysis
  • Technical
    • Architectural objectives
  • Risk
    • Sensitivity analysis
    • Monte Carlo simulations

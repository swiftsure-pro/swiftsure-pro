Artifact: Resources List
This artifact captures data about an organization's consolidated resources capacity.
Domains: Portfolio Management, Product Portfolio Management
Purpose
To provide information about the organization's resources to enable analysis of the total resources demand and supply of the organization.
Relationships
RolesResponsible:
  • Resource Manager
Modified By:
  • Resource Manager
TasksInput To:
  • Optimize Portfolio

  • Consolidate Supply
Output From:
  • Consolidate Supply
Description
Main DescriptionThis artifact captures a list of available resources (considering time, team distribution, roles, efficiency, and so forth) that can be used to supply the work demand.
Key Considerations

Depending on the size of the organization, the scope and the tool support of the portfolio management solution, it might be sufficient to manage the resources list from an overall level, such as not managing individual resources, but keeping the resources list on pool level only.

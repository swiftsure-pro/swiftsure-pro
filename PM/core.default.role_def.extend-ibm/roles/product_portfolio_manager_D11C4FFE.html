Role: Product Portfolio Manager
This role manages, reviews, and helps make decisions about the portfolio and its components.
Role Sets: Portfolio Management

Modifies
  • Components List

  • Portfolio Constraints and Metrics Document
  • Portfolio Recommendations
  • Portfolio Structure Document
    Main Description

    This role is charged with the following responsibilities:

    • Have continuous oversight of the contents within a product portfolio
    • Conduct a periodic review of the performance (and conformance to expectations) of products within the portfolio
    • Ensure that data about each of the products in the portfolio is collected and analyzed
    • Enable periodic decision-making about the future direction of the products and the entire portfolio

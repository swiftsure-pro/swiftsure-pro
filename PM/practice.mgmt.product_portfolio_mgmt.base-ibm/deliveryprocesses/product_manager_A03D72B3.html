Role Descriptor: Product Manager
This role deals with the planning or marketing of a product (or products) at all stages of the product lifecycle.

Main Description

This role generally guides the coordination of what is included in each version of a product as part of an overall product strategy. This role's responsibilities span many activities, from making strategic to tactical decisions, which vary based on the organization's size and the structure of the company.

This role is also responsible for gathering user requests from different stakeholders and deciding which user requests are to be implemented in a particular release of the product. This role is responsible for planning releases and managing the discussions with the stakeholders.

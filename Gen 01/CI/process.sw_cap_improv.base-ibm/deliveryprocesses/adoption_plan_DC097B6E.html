Main Description

This document serves as the implementation plan for rolling out best practices and tools to the organization. It provides schedule and resource estimates, pilot project training and mentoring estimates for deployment in support of the key business drivers for the organization.

This artifact describes all important or critical decisions regarding the proposed implementation of practices and tools. It describes the overall approach as well detailed implementation plans for deployment.

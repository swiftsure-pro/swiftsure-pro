Task Descriptor: Develop Method Library User Guidelines
This task describes how to develop user guidelines for method library use.

Steps
Determine the scope of the user guideline

Typically, this guideline will include a "how-to" section for using the method library and the method assets.  It may also include an overview of the available assets, guidance for installing/downloading method configurations and related assets, and guidance for using the authoring tools that helps to extend/tailor method elements.  It may include policies for what changes are permitted and what approval process may be required

Write the guideline
Deploy the guideline

Choose where the guideline can be found for users: packaged with a deployed library or available online for all relevant users.

Task Descriptor: Define Enterprise Tool Configurations
This task describes how to define configurations of tools for use by the enterprise.
Based on Method Task: Define Enterprise Tool Configurations
Relationships
RolesPrimary:
  • Tool Specialist
Additional: Assisting:
Main Description

Good processes should be supported by good tools that help make the process run smoothly.  Enterprise tool configurations may be hosted for use by projects, or simply be documented for someone on the project to use when configuring the tool.

Steps
Determine the enterprise needs in term of tool support
Identify the kind of tools required and how they can help to achieve enterprise goals.  This may be part of a larger enterprise effort.
Determine the scope and usage policy of the tools
Identify who needs to use which tools and how these tools will be made available to their users.  For example, will the tool be deployed for projects to use, or do projects need to install something themselves.
Setup and configure enterprise tools
For tools that are provided by the enterprise, they need to be installed and configured.  For tools which are installed by someone else, you need to setup and configure the tool in order to confirm that the configuration works as expected.
Analyze and revise enterprise tool scope and policy
Analyze the effectiveness of tools installed and determine if changes are necessary to maximize the benefit of these tools.

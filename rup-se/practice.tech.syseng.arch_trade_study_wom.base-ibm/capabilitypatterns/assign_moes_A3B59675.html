Task Descriptor: Assign MoEs to Candidate Solutions
This task focuses on assigning Measures of Effectiveness (MoE) values to each of the candidate solutions.

Main Description

The Measure of Effectiveness (MoE) describes how effective a solution candidate is for a particular assessment criterion. The MoE is a normalized value computed using the utility curve and the nominal value specified for the solution candidate.

The nominal values are typically determined from equipment specifications or derived from calculations based upon the relevant solution.

The assignment of MoEs to each candidate solution is a prerequisite for the identification of the preferred solution.

Steps
Select an assessment criterion
There will be a utility curve associated with each assessment criterion. Select an assessment criterion and perform the following steps.
Determine the nominal values for each solution candidate
For the subject assessment criterion, determine the nominal values for each solution candidate. This information may be taken from equipment specifications, measurements or analysis.
Compute MoE for each solution candidate
Convert the nominal values for each solution candidate to the associated Measure of Effectiveness by applying the utility curve for the subject criteria.
Repeat analysis for each assessment criterion
Repeat the previous three steps for each assessment criterion to complete the assessment.
Update the Trade Study Report
Update the relevant section of the Trade Study report to capture the matrix of MoEs.

Work Product Descriptor (Artifact): System Architecture Model
A system model is a model of the system, or a part of the system, that defines the main elements, their structure, relationships and behavior. The system model is typically maintained in a modeling tool.

Main Description
The System Architectural Model is a hierarchical structure of System Blocks, behavior and delegation ports for each block, interfaces definitions for each port, and state-chart diagrams specifying the state-based behavior for each leaf system block.

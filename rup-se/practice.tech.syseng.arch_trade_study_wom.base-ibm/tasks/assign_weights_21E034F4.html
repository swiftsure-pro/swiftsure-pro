Task: Assign Weights to Criteria
This task focuses on establishing weighting factors for each criterion.
Disciplines: System Engineering Requirements
Purpose
In order to prioritize the assessment criteria and to allow direct comparison between them, each criterion is assigned a weighting factor.
Relationships

      Main Description

      Not all assessment criteria are equal. Some are more important than others. Assessment criteria are weighted according to their relative importance to the overall solution. The weighting factors are normalized to add up to 1.0. This task should be performed in collaboration with stakeholders and relevant domain experts.

      Steps
      Assign relative weights to the assessment criteria
      Assign a weight to each assessment criterion based upon its relative importance such that the sum of all weights equals 1.0.
      Update the Trade Study Report
      Capture the assessment criteria weights in the Trade Study Report.

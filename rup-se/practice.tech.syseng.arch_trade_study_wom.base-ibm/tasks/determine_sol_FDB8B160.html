Task: Determine Solution
This task focuses on determining the optimum solution for a selected aspect of the system.
Disciplines: System Engineering Requirements
Purpose
By means of the Weighted Objectives Analysis the optimum solution for a chosen area of the system is determined.

      Main Description

      The determination of a preferred solution is performed by means of Weighted Objectives calculations. In this analysis the MoE values for each of the assessment criteria are multiplied by the appropriate assessment weight. The weighted values for each alternative solution are then added to obtain a total score for each solution. The solution with the highest score is selected as the implementation for that particular aspect of the system.

      Steps
      Select candidate solution
      The following steps will be performed for each candidate solution in order to determine the preferred solution. Select a candidate solution and perform the following steps.
      For each assessment criterion calculate the weighted value
      The weighted value of each assessment criterion is the product of the weight for that criterion and the corresponding MoE value.
      Determine the total weighted score of the solution
      The total weighted score of the candidate solution is the sum of its associated weighted values.
      Repeat steps for each candidate solution
      Repeat the previous three steps for each candidate solution.
      Determine the solution
      The optimum solution is the one with the highest total weighted score.
      Update the Trade Study Report
      Capture the results of the work completed in the Trade Study Report.

Task: Refine Operations
This task refactors and refines the previously identified operations.
Disciplines: System Engineering Architecture
Purpose
To refactor and refined the system operations which have been previously defined.
Relationships
RolesPrimary Performer:
  • System Architect
Additional Performers:
InputsMandatory:
  • System Operation
    		Optional:
    • None
    Outputs
    • System Operation
      Steps
      Analyze system operations for overlaps
      Look for identical or quasi-identical system operations and collapse them into a minimum set of operations.
      Refine system operations
      Look for operations which need to be described in more detail. Use reflexive operations if needed.

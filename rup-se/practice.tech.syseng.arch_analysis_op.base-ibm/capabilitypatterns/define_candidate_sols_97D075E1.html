Task Descriptor: Define Candidate Solutions
This task focuses on identifying possible solutions for a chosen viewpoint.
Based on Method Task: Define Candidate Solutions

  • None
Outputs
  • System Architecture Structure Diagram
  • System Use Case
Steps
Select a viewpoint
Select the system aspects (reflected in a viewpoint) for which an architectural analysis will be performed.
Create an initial decomposition
Depending on the viewpoint type, decompose the system into its viewpoint/view-specific elements and outline the significant collaborations that realize the main system's operations.
Look for alternatives
Based on your experience, on the existent and/or available assets, non-functional requirements, and other constraints, define alternate decompositions into main elements and their associated collaborations.
Select candidate solutions
Identify those solutions that will be considered as candidates for a detailed trade study analysis.
Create Trade Study Report
Capture the selected candidate solutions in the Trade Study Report.

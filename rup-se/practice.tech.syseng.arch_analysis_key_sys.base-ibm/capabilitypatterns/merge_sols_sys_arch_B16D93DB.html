Task Descriptor: Merge Solutions into System Architecture
This task focuses on merging the individual solutions for each key system function to define the system equipment breakdown structure.

Main Description

The solutions selected for each key system function are merged to define the equipment breakdown structure. It is assumed that the initial key system functions were independent. Thus, the final merged solution is the preferred solution based upon the assessment criteria for the complete architecture. It will be the basis of the subsequent architectural design activities. These design decisions are captured in the Trade Study Report along with any resulting design constraints.

Steps
Capture the results
Update the Trade Study Report to capture the preferred equipment breakdown structure and any resulting design constraints.

Task Descriptor: Link Stakeholder Requirements to System Requirements
This task presents some of the key aspects of maintaining the traceability between the stakeholder requirements and the system requirements.

Main Description

Establish traceability from each System Requirement to one or more requirements in the Stakeholder Requirements Specification to ensure that:

  • All System Requirements are derived from Stakeholder Requirements
  • All Stakeholder Requirements are satisfied by one or more System Requirements.

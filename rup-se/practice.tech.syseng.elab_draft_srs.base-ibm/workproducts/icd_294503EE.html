Artifact: Interface Control Document
This artifact specifies the logical and physical interfaces between a system element and its environment.
Domains: Systems Requirements
Purpose
To specify the logical and physical interfaces between a system element and its environment so that these interfaces can be communicated and agreed between relevant parties to ensure that the element will function properly in the intended environments.

Description
Main Description

This artifact includes specifications for all thermal, mechanical, electrical, optical and chemical interfaces between the element and its operational, maintenance, and test environments.

The system element may be the entire system, in which case the interfaces represent the external system interfaces.  Alternatively, the system element may be a segment, subsystem, unit, or component of an enclosing system.  In which case, the interfaces represent the internal interfaces of the enclosing system.

There will be multiple instances of this artifact. In particular, there will always be an instance of this artifact that specifies the external interfaces, which may include interfaces required for testing and maintenance as well as the operational interfaces, of the system.  This ensures that the system will be interoperable within the operational (and test and maintenance) environment.

Additional instances of this artifact will be required to specify the internal interfaces for each system element.  This ensures that the system element will be interoperable within the enclosing system.

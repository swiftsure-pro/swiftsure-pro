Work Product Descriptor (Artifact): Issue
This artifact is a work item type used to track the work associated with managing an issue.
Based on Method Work Product: Issue
Main Description

An issue work item is provided to track status of exceptions, problems, or issues. The team handles exceptions and problems, and captures a list of issues in the Iteration Plan or Work Items List.

Artifact: Task
This artifact is a work item type used to track the work associated with a development task that is performed as part of an iteration.

Description
Main Description

This artifact is contained by the work items list, which keeps track of all work to be performed in a project. During iteration planning, tasks are added to the work items list and are signed up for by team members who will take ownership of performing each task until it is closed.

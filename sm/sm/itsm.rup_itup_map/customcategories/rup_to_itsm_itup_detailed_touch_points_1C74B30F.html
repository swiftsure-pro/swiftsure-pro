Main Description

The Quick Reference to Integrated Touch Points provides the links to method elements that are specific to understanding the integration of IT software development and IT operations.

The Touch Points by ITUP Process Categories provides links to the method elements covering the integration of the IBM Tivoli Unified Process (ITUP) processes with the RUP grouped by the set of IT Processes within an IT organization.

The Touch Points by RUP Discipline provides links to the method elements describing the RUP to ITUP integration grouped by the RUP disciplines.

The Touch Points in Alphabetical Order provides a mapping of the method elements that describe the RUP to ITUP integration touch points.

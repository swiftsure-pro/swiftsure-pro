Term Definition: Configuration Management Database (CMDB)
Main Description

The CMDB is a support tool and defines relationships between Configuration Items (CIs). That is, whether a CI is part of, is connected to, joins another CI. The CMDB also contains information about incidents, problems, known-errors, changes, and corporate data about employees, suppliers, locations, business units and information on back-up copies of all CIs.

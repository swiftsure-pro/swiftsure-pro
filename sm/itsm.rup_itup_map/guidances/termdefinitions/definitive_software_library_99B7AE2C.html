Term Definition: Definitive Software Library (DSL)
Main Description

A secure storage area in which the authorized versions of all software Configuration Items are stored and protected. This one storage area, in reality, consists of one or more software libraries or file-storage areas that should be separate from development, test, or live file-store areas. It contains the master copies of all controlled software in an organization. The DSL should include definitive copies of purchased software as well as software developed on-site. Master copies of controlled documentation for a system will also be stored in the DSL in electronic form. These may include license documents, software instructions, and other software documentation.
